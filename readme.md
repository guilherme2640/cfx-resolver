<h1 align="center">CFX Resolver</h1>


<h2>Requisitos</h2>
<hr>
<a href="https://www.ruby-lang.org/en/">Ruby 3.x</a>


<a href="https://bundler.io/">Gem Bundler</a>
<h2>Modo de uso:</h2>
<hr>

<ol>
<li>Clone o projeto:


```shell
$ git clone https://github.com/guilherme2640/cfx-resolver
```
</li>
<li>Instale as gems:


```shell
$ bundle install
```
</li>
<li>Rode o programa:


```shell
$ ruby main.rb
```
</li>
</ol>
